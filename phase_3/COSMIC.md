# COSMIC Function Points – EventForge
**Обозначения COSMIC FP:**  
- **E** – Entry (ввод данных пользователем)  
- **R** – Read (чтение или выбор данных из системы)  
- **X** – Exit (вывод данных пользователю)  
- **W** – Write (запись данных в систему)
