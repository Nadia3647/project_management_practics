## COSMIC Function Points

По результатам функционального анализа системы по методике COSMIC получено:

Итоговая оценка: **34 FP**

Оценка учитывает:
- ввод пользовательских данных (Entry),
- чтение данных (Read),
- запись данных в систему (Write),
- вывод информации пользователю (Exit).

Методика подтверждает, что система обладает насыщенной бизнес-логикой, элементами аналитики и AI-компонентами.

Итоговые параметры расчёта:

- Unadjusted Actor Weight (UAW) = 36
- Unadjusted Use Case Weight (UUCW) = 115  
- Technical Complexity Factor (TCF) = 1.01  
- Environmental Complexity Factor (ECF) = 0.73  


Итоговая оценка проекта: **примерно 112 Use Case Points**

---

## Оценка трудоёмкости и сроков разработки

В практике проектного управления применяется следующая зависимость:

1 UCP ≈ 20–22 человеко-часа.

Тогда:

- 112 × 20 = 2240 человеко-часов  
- 112 × 22 = 2464 человеко-часов  

Общая трудоёмкость проекта: **примерно 2300–2500 человеко-часов**.

## Расчёт сроков реализации

Средняя загрузка одного специалиста составляет 160 человеко-часов в месяц.

Тогда ресурс команды из 6 человек:

6 × 160 = 960 человеко-часов в месяц


С учётом встреч, итеративности разработки, тестирования и коммуникаций эффективная загрузка составляет около 80%:

60 × 0.8 ≈ 768 человеко-часов в месяц


Оценка сроков:

2300 / 768 ≈ 3 месяца


Итоговый срок реализации проекта: **около 3 месяцев**
