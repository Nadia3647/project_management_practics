# Оценка проекта с использованием нечётких критериев

## Критерии оценки

1. **C1: Уровень конкуренции в отрасли** (стоимостной — чем меньше, тем лучше)
2. **C2: Срок окупаемости проекта** (стоимостной — чем меньше, тем лучше)
3. **C3: Стоимость реализации МУР (MVP)** (стоимостной — чем меньше, тем лучше)
4. **C4: Количество потенциальных клиентов** (выгодный — чем больше, тем лучше)

## Детализация критериев

### C1: Уровень конкуренции в отрасли
**Чем меньше конкуренции, тем лучше**

| Уровень | Описание | Нечёткая оценка |
|---------|----------|-----------------|
| Хорошо  | Почти нет прямых конкурентов, низкая насыщенность рынка | (1, 3, 5) |
| Средне  | Есть несколько заметных конкурентов, рынок умеренно насыщен | (3, 5, 7) |
| Плохо   | Высокая конкуренция, множество игроков, рынок перенасыщен | (5, 7, 9) |

### C2: Срок окупаемости проекта
**Чем меньше срок, тем лучше**

| Уровень | Описание | Нечёткая оценка |
|---------|----------|-----------------|
| Хорошо  | Окупаемость до 6 месяцев | (1, 3, 5) |
| Средне  | Окупаемость от 6 месяцев до 1,5 лет | (3, 5, 7) |
| Плохо   | Окупаемость более 1,5 лет | (5, 7, 9) |

### C3: Стоимость реализации МУР (MVP)
**Чем меньше стоимость, тем лучше**

| Уровень | Описание | Нечёткая оценка |
|---------|----------|-----------------|
| Хорошо  | Низкая стоимость, реализация силами 1–2 человек за короткий срок | (1, 3, 5) |
| Средне  | Средние затраты, требуется небольшая команда и несколько месяцев | (3, 5, 7) |
| Плохо   | Высокие затраты, привлечение сторонних ресурсов, длительная разработка | (5, 7, 9) |

### C4: Количество потенциальных клиентов
**Чем больше клиентов, тем лучше**

| Уровень | Описание | Нечёткая оценка |
|---------|----------|-----------------|
| Хорошо  | Широкая аудитория, массовый спрос | (1, 3, 5) |
| Средне  | Ограниченный, но стабильный сегмент рынка | (3, 5, 7) |
| Плохо   | Узкая ниша, небольшое количество потенциальных клиентов | (5, 7, 9) |
